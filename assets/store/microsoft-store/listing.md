## Store description

Requires the Microsoft Visual C++ Redistributable. Microsoft Store installs this dependency automatically.
IPTVS Player is a media player for IPTV services and playlists you already have permission to access.

Add a Stalker portal, Xtream account, or M3U playlist and browse live TV,
movies, and series in one interface. Provider support determines which
channels, programmes, catch-up streams, and media details are available.

Features include programme-guide browsing, catch-up playback, favourites,
local profiles, Continue Watching, HDR playback on supported hardware, and
keyboard, mouse, and remote-friendly navigation. Optional cloud sync lets you
manage profiles and source configurations from a web panel and pair devices
without typing an account password on a TV.

No login or paid IPTV credentials are required to explore the app. Create a
local profile and the built-in Demo catalogue opens automatically with public
playback fixtures, open movies, generated guide data, and browsable series.

IPTV subscriptions, channels, playlists, and media are not included. You must
provide your own lawful source. IPTVS Player is not affiliated with or endorsed
by IPTV providers, broadcasters, or content owners.

## Reviewer notes

No credentials are required. On first launch, select **Add profile**, enter any
local profile name, and select **Create**. The new profile opens directly into
the preselected Demo source.

The Demo source provides Live, Movies, and Series tabs; generated now/next and
past programme-guide rows; simulated catch-up on the two open-movie channels;
favourites; movie detail metadata; complete season/episode browsing; and public
HLS, fMP4, HEVC, and MP4 playback fixtures.
