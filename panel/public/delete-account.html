Delete cloud account

IPTVS Player

IPTVS Player does not require an account. These instructions apply only if you created an optional cloud-panel account using an email sign-in link.

Delete the account yourself

  1. Sign in to the IPTVS cloud panel.
  2. Open Account.
  3. Select Delete cloud account.
  4. Type DELETE when asked to confirm.

This permanently removes the cloud account, email-linked account identity, profiles, source configurations and credentials, metadata settings and API keys, favourites, device labels, and pairings. The action cannot be undone.

If you cannot sign in

Email gchofficial@gmail.com from the address used for the cloud account and request “IPTVS cloud account deletion.” Identity may need to be confirmed before deletion. Do not include provider credentials or playlist URLs.

Data that remains on devices

Deleting the cloud account cannot remotely erase local profiles, caches, favourites, or playback history already stored on disconnected devices. Clear IPTVS Player's storage or uninstall it on each device to remove that local data.
